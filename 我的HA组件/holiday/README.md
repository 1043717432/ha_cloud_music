 # 判断是否节假日

 ## 如何使用

> 安装

 将本项目【holiday文件夹】复制到HA的custom_components文件夹中

> 配置

 然后在configuration.yaml中配置以下内容

```

binary_sensor:
  - platform: holiday

```
